# Monocai Dark Theme

A VS Code theme based on the Monocai Dark color scheme from IntelliJ IDEA, using the structure from TmTheme-Editor.

## Installation

1. Copy this extension folder to your VS Code extensions directory
2. Reload VS Code/Cursor
3. Select "Monocai Dark" from the theme picker

## Color Scheme

This theme features:
- Dark background (#191a1b)
- High contrast syntax highlighting
- Colors ported from Monocai Dark IntelliJ IDEA theme
